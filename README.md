# Meta Ads Performance Dashboard v2 (Next.js)

Este é o novo dashboard de performance do Meta Ads, totalmente reconstruído em **Next.js 15**, **React 19** e **Node.js**, focado em estética premium e performance state-of-the-art.

## 🚀 Tecnologias
- **Framework**: Next.js 15 (App Router)
- **Estilização**: TailwindCSS 4 (Bento Design System)
- **Animações**: Framer Motion
- **Gráficos**: Recharts
- **Ícones**: Lucide React
- **API**: Meta Graph API via Node.js

## 💎 Design System
- **Cantos Arredondados**: Estética baseada em `rounded-3xl` para um look moderno e amigável.
- **Glassmorphism**: Efeitos de transparência e desfoque de fundo para profundidade visual.
- **Tipografia**: Gate (Primária) e Poppins (Secundária).

## 🛠️ Instalação
1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Configure as variáveis de ambiente no `.env` (baseado no `.env.example`).
4. Rode o projeto: `npm run dev`.

## 📂 Estrutura de Pastas
- `src/app`: Páginas e rotas da aplicação.
- `src/components`: Componentes UI reutilizáveis.
- `src/lib`: Utilitários e cliente da API do Meta.
- `legacy-v1-python`: Código original em Python para referência.
